# 🎓 Student Management System (Frontend)

A modern **Student Management System UI** .  
This project focuses on clean design and smooth user experience for managing students and courses.

---

## 🚀 Features

- 👨‍🎓 Student List UI
- ➕ Add Student Form
- ✏️ Edit Student (UI)
- 🗑️ Delete Student (UI)
- 🔍 Search & Filter
- 📱 Responsive Design
- 🎨 Clean Dashboard Layout

> ⚠️ Note: This is **frontend only** (no backend/API connected yet)

---

## 🛠️ Tech Stack

- React (Vite)
- JavaScript (ES6+)
- Axios (optional for future API)
- React Router
- CSS / Tailwind (if you used it)
